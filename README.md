# Lottery CLI

> A CLI that will make a bettor's life easier.

![Example CLI running](img/example.gif)

### Installing

```
$ npm install -g lottery-cli
```

### How to use

```sh
lottery --help

  Usage: lottery <command> [flags]

  where <command> is one of:
      megasena

  lottery help <command>  quick help on <command>
```
